Unix Command

  • Mac OS X is built on top of Darwin, a version of unix based on BSD 4.4, FreeBSD and the Mach microkernerl
  • Unix is case sensitive

General

  • date
  • who/who am i/finger/w
  • Find out what shell you are using with echo \(SHELL or ps \)$
  • For zsh, use bindkey to find out all the key bindings
  • Use man <command> or ls --help for more help. Press q to exit out of man command
  • Groups - To see who current user is a group of.

Less

  • Good for txt files and code files
  • Spacebar to move down
  • h to get help
  • q to quit

Managing Files

  • ls -ld directory/filename
  • Results: drwxrwxrwx@ 5 jamietan staff 160 Feb 11 17:33 Ex_Files_Learning_Vim
    • d - directory
    • Permissions for owner, group, and others
    • @ â†’ Indicates extended attributes (you can check them with ls -l@
    • (5) shows the number of hard links to this directory (typically includes the directory itself, its parent (..), and subdirectories).
    • The third column (jamietan) indicates the owner of the directory.
    • The fourth column (staff) shows the group the directory belongs to.
    • The fifth column (160) represents the size of the directory entry in bytes (not the total size of its contents).
    • The sixth column (Feb 11 17:33) shows the last modification date and time of the directory.
    • The last column is the name of the directory.
  • chmod permissions filename
  • u user g group o others a all / + add - delete = exactly / r read w write e execute
  • chmod u+w filename / chmod ug=rw *

Manage Process

  • Find process ps aux | grep process
  • Kill process kill -9 process
  • Check key bindings in bash/zsh with stty -a
  • Look for susp = ^z (this is the command to suspend processes)
  • fg = resume in foreground
  • bg = resume in background

Manage Jobs

  • Find jobs that are running jobs
  • Kill job kill %[jobnumber]

Install Brew on Mac

  • /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
  • brew --version
  • Run these commands in your terminal to add Homebrew to your PATH: 
    echo >> /Users/jamietan/.zprofile
echo 'eval "$(/opt/homebrew/bin/brew shellenv)"' >> /Users/jamietan/.zprofile
eval "$(/opt/homebrew/bin/brew shellenv)"
  • Run brew help to get started
  • Further documentation: https://docs.brew.sh

Install FTP

  • Check for version: which ftp
  • brew install inetutils
  • Alternative ftp: brew install lftp

Install Lynx

  • which lynx or lynx --version
  • brew install lynx
  • lynx "https://website"
  • Spacebar to move forward or b to back
