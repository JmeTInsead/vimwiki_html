VIM Commands

Basic Navigation

h → Move left l → Move right j → Move down k → Move up w → Move to the next word b → Move to the beginning of the previous word 0 → Move to the beginning of the line ^ → Move to the first non-blank character of the line $ → Move to the end of the line gg → Move to the beginning of the file G → Move to the end of the file Ctrl-d → Move half a screen down Ctrl-u → Move half a screen up Ctrl-f → Move one full screen down Ctrl-b → Move one full screen up

Insert Mode

i → Enter insert mode before the cursor I → Insert at the beginning of the line a → Insert after the cursor A → Insert at the end of the line o → Open a new line below O → Open a new line above Esc → Exit insert mode

Editing

x → Delete the character under the cursor X → Delete the character before the cursor dd → Delete (cut) the current line yy → Copy (yank) the current line p → Paste after the cursor P → Paste before the cursor u → Undo Ctrl-r → Redo r<char> → Replace the character under the cursor R → Enter replace mode :s/oldword/newword or :s/oldword/newword/gc (g switch to replace all)

Searching

/pattern → Search forward for "pattern" ?pattern → Search backward for "pattern" n → Repeat last search in the same direction N → Repeat last search in the opposite direction

  • → Search forward for the word under the cursor
  • → Search backward for the word under the cursor

Saving & Exiting

:w → Save file :q → Quit :wq or ZZ → Save and quit :q! → Quit without saving :w filename → Save as a new file

Working with Multiple Files

:e filename → Open a file :bn → Next buffer :bp → Previous buffer :bd → Close (delete) current buffer :sp filename → Split window and open file :vs filename → Vertical split and open file Ctrl-w w → Switch between splits

Creating Marks

mA (global mark) ma (local mark). Call a mark 'A

  • Add character for multiple lines

## Front

  1. Navigate to the start of the place for the first line.
  2. Use Ctrl V (Visual Block) and J/arrow key to highlight several lines
  3. Use Shift + I to indicate you want to enter with the cursor.
  4. Click Esc and all lines will be populated.

## Back

  1. Navigate to the end of the first line where you want to insert the symbol.
  2. Use Ctrl +V (Visual block) and highlight multiple lines
  3. Use Shift + A to indicate you want to enter after the cursor. Click esc and all lines will be populated.
