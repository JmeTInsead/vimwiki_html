In Vim, when you use Shift + G (G) to jump to the end of the document, the cursor lands on the last character of the last line, rather than after it.

To start inserting after the last character of the last line without using arrow keys, use:

A → Moves the cursor to the end of the line and enters Insert mode. o → Creates a new line below and enters Insert mode. If you specifically want to be at the absolute end of the document, the best approach is:

Press G to go to the last line. Press A to append after the last character.
