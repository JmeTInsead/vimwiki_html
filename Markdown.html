## Basic Style

  • Create emphasis or *bold*
  • Code Tag using backtick x2
  • If you tab/add 4 spaces in markdown, it puts it into a box and uses <pre><code
  • Blockquote using greater than sign >. Blockquote inside another block quote would be >>
  • Draw line using --
  • URL <url link>
  • Reference Links [link][label]
  • [label]: https://www.flyingcape.com.sg
  • Image Link ![Gummy Bears)(http:/imageURL)]
  • Think about how you can make an image clickable by combining both.

Bonus Tip

  • Use imgur to host images. --------------------------------------
