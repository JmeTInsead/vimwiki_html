Setting up Plugins

  • Install vim-plug
    • curl -fLo ~/.vim/autoload/plug.vim --create-dirs https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim
  • Verify installation by checking the file exists: ls ~/.vim/autoload/plug.vim
  • Add Plugin configuration into .vimrc (Refer to configuration file below)
  • Run :PlugInstall
  • Run configuration file in vim - :source ~/.vimrc

NERDTree configuration

  • Add NERDtree to vim configuration file 
    call plug#begin('~/.vim/plugged')
" Install NERDTree
Plug 'preservim/nerdtree'
call plug#end()

" Open NERDTree automatically when Vim starts
autocmd VimEnter * NERDTree
    " Close Vim if NERDTree is the last window open autocmd BufEnter * if winnr('$') == 1 && exists('b:NERDTree') | quit | endif
