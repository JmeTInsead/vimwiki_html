Setting up sync to Github

  1. Create Github Rep - No Read Me and Private Repository
  2. Initialise Git in Vimwiki Directory
    • cd ~/Documents/vimwiki #navigate to the directory
    • Add the Remote Repository: git remote add origin https://github.com/JmeTInsead/vimwiki-notes.git
    • Add all files to Git: git add .
    • Commit the changes: git commit -m "Initial Vimwiki sync"
    • Push to Github:
      • git branch -M main #Rename the branch to 'main' if needed
      • git push -u origin main
  3. Ensure that you are connected to Github using PAT
    • Developer Access and provide full access to repo.
    • Update Remote URL to use PAT: git remote set-url origin https://yourusername:yourtoken@github.com/yourusername/vimwiki-notes.git
    • git push origin main #Push your vimwiki again
  4. Automate Syncing with a Shell Script
    • Create a Sync Script in folder i.e. vim ~/Documents/vimwiki/sync.sh
    • Paste the following script 
      #!/bin/bash
cd ~/Documents/vimwiki
git add .

Retrieving from Github

  • Check that you have git installed: git --version
  • Make the directory where you will store the cloned github repo.
  • Clone the repository: git clone <repository-url> <localfoldername>
  • Enter the local directory where you cloned it and pull the latest code: git pull origin main
  • Set up .vimrc to point to this folder
